# -Stock-Data-Analysis-Dashboard-Power-BI-Project
This project focuses on analyzing stock market data using Power BI to visualize key metrics such as trading volume, price trends, and yearly price ranges.  The dashboard provides insights into stock performance through interactive charts and KPI cards.

# Stock Data Analysis Dashboard (Power BI)

## Tools & Technologies

* Power BI
* Excel / CSV Dataset
* Power Query (Data Cleaning)

## Dashboard Features

* KPI Cards displaying:

  * Average 52 Week High
  * Average 52 Week Low
  * Total Volume
  * Total Open Price
  * Total Close Price

* Interactive filters for:

  * Day
  * Month
  * Quarter
  * Year

* Data visualizations including:

  * Quarterly comparison of 52W High vs Low
  * Daily Open vs Close price trends
  * Monthly trading volume analysis
  * Day-wise comparison of stock metrics

## Key Insights

* Identified trends in stock price movements
* Analyzed trading volume patterns across months
* Compared quarterly stock performance
* Observed daily price fluctuations

## Learning Outcomes

* Data cleaning using Power Query
* Building interactive dashboards in Power BI
* Data visualization and storytelling
